## 인덱스 (Index)
## 슬라이싱(Slicing)
### 슬라이싱 예시
```python
my_str = 'hello'
```
```python
my_str[1:] # 'ello'
my_str[:3] # 'hel'
my_str[::2] # 'hlo'
my_str[::-1] # 'olleh'
my_str[1:3] # 'el'
my_str[1:4:2] # 'el'
my_str[5:2:-1] # 'ol'
```
